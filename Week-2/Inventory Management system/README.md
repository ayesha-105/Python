A simple OOPs based command-line project to manage inventory items using file handling (JSON).
This app allows you to Add, Update, Delete, and View inventory items.

**Features**
- Add new inventory items
- Update existing item details
- Delete an item by ID
- View all inventory items
- Date added is automatically saved according to current date
- Data is stored in a JSON file

**Concepts Used**
- Python user-defined functions
- JSON file handling
- Classes and objects
- Loops and conditionals
- Exception handling
